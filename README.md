# Build

The game was created with **Unity 2018.3.2f1**.

 1. Install Unity
 2. Clone this repository
 3. Enjoy the game

# Controls

- Arrow Keys / WASD to move
- Space / Return to interact
- ESC to exit

| Action        | Bind          |
| ------------- |:-------------:|
| Move Left | A / Left Arrow |
| Move Right | D / Right Arrow |
| Interact | Space / Return |
| Exit | Escape |
